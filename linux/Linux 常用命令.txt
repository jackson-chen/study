----------Linux常用命令------------
shutdown -h now | halt  关机
shutdown -r now | reboot   重启  
ls     查看目录
	-a  查看所有文件
	-l  长格式
	-h  便于人类查看文件大小
	-S  文件从小到大排列  
	-r  从小到大排序
mkdir -p  name   创建目录
rmdir -p  name   删除目录
man  命令(man ls)  查看其他命令的帮助信息
cd   pathname   切换目录 
pwd  查看当前所在目录	 
touch  filename   创建文件
cat     filename  显示文件内容
     -n  显示行号

more  filename  一屏的方式显示文件内容
      按回车  一行行的显示
      按空客  一页页的显示

lesss filename  和more差不多一样

head  filename 显示文件头几行
tail  filename 显示文件最后几行
cp    来源文件   目标文件   复制文件
    -r  递归复制
mv    来源文件   目标文件       移动文件
rm    删除目录和文件
    -r  递归删除
    -f  删除不确认

which   命令  查看命令
whereis 名称  特定的几个目录查找
find  目录  -name(-user)  filename('filename*','file??')
     -user 表示查找指定用户创建的文件
     'file*' 'file?' 表示模糊匹配

Linux目录解析
 /bin   命令存放目录，普通用户、超级用户使用的命令
 /sbin  命令存放目录，超级用户使用的命令
 /boot  启动目录
 /dev   设备文件存放目录
 /etc   配置文件存放目录
 /lib   函数库存放目录
 /home  普通用户家目录
 /mnt   系统挂载目录
 /media 媒体设备挂载目录 
 /opt   第三方软件存放目录
 /root  超级用户家目录
 /tmp   临时目录
 /proc  放置数据到内存
 /srv   服务存放数据目录
 /usr   系统软件资源目录
 /var   系统相关文档内容目录

【权限】

符号   权限       文件          目录
r      读权限     查看文件     查看目录
w      写权限     修改文件     修改目录内容
x      执行权限   执行文件     进入目录


修改权限：  	chmod -R  xyz  文件
			
			方案一：
			x:所有者权限, y:所属组，z:其他用户
			r:4 y:2  z:1    数字总和即权限

			方案二：
			x 角色 ：   u 代表所有者  g所属组  o其他用户  a所有用户
			y 设置 ：   + 代表增加   - 减少   = 设置



【角色】

		      所有者       所属组     其他用户
文件/目录      rwx          rwx        rwx


文件类型 
 - :文件    d:目录   l:链接

修改所有者 chown -R 用户名  文件名
修改所有者和组：chown -R 用户名:组名 文件或目录
修改所属组:  chgrp  组名   文件名


【压缩】

压缩的类型
.gz      gzip压缩格式的文件
.bz2     bzip2压缩格式的文件
.xz      xz压缩格式的文件
.zip     zip压缩格式的文件
.tar     tar打包的文件
.tar.gz  打包并用gz压缩的文件
.tar.bz2
.tar.xz

tar [-ctxzjJvf]  压缩文件  [源文件]
c   打包压缩
t   查看内容
x   解打包解压缩
z   使用gzip方式压缩
j   使用bzip2
J   使用xz
v   显示过程
f   指定压缩包名
举例：
	tar -cvf   mytar.tar  a.txt  b.txt  
	tar -czvf  mytar.tar.gz a.txt b.txt
	tar -tvf   mytar.tar
	tar -tzvf  mytar.tar.gz
	tar -xvf   mytar.tar
	tar -xjvf  mytar.tar


【软件安装】
1.源码安装
	下载源码包(curl,wget)
	解压(tar)
	进入该目录(cd)
	编译前配置(./configure)
	编译(make)
	编译安装(make install)
	删除 make clean  然后直接删除目录


注意：
	在编译安装的是否可能会依赖其他的软件包，需要去安装他们

举例：
	 curl -O  ftp://mama.indstate.edu/linux/tree/tree-1.8.0.tgz
	 tar -xzvf  tree-1.8.0.tgz
	 cd  tree-1.8.0.tgz
	 make install


2.rpm包安装
rpm  -ivh  软件包
  -i 安装  -v 显示详细信息  -h  显示进度

查询软件包是否安装 rpm  -q  安装包
查询软件包信息   rpm  -qi  安装包 
查询软件包的安装位置  rpm -ql   安装包
卸载包   rpm -e 安装包




3.yum安装管理rpm包

查询可以安装的软件包  yum list  名称
安装   yum  -y  install  软件包      说明: -y 自动回答yes
更新   yum  -y  update   软件包
卸载   yum  -y  remove   软件包             



4.总结以上几种安装方式
源码包可以在所有平台安装
rpm centos系统可以使用
yum 自动下载依赖，省时省力
